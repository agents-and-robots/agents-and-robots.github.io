One Column (Eine Spalte)

Kurzbeschreibung

Ein einfaches einspaltiges Layout, das häufig für Artikel, Blogbeiträge oder andere Inhalte verwendet wird, bei denen der Fokus auf dem Fließtext liegt.

Beispiel

Überschrift

Dies ist ein Beispieltext für das einspaltige Layout.

HTML und CSS

<div style="border: 1px solid #ccc; padding: 16px; max-width: 600px; margin: 0 auto;"> <h2>Überschrift</h2> <p>Dies ist ein Beispieltext für das einspaltige Layout.</p> </div>

Beschreibung

Das One Column Layout ist ideal für Inhalte, die linear gelesen werden, wie z. B. Artikel, Blogbeiträge oder lange Texte. Durch die zentrale Anordnung wird der Fokus des Lesers auf den Inhalt gelenkt. Es ist einfach umzusetzen und auf allen Bildschirmgrößen flexibel anwendbar.
