Responsive Flexbox Layout

Kurzbeschreibung

Ein flexibles und responsives Layout, das mit Flexbox erstellt wurde. Es passt sich dynamisch an verschiedene Bildschirmgrößen an.

Element 1
Element 2

HTML und CSS


<div class="flex-container">
    <div class="flex-item">Element 1</div>
    <div class="flex-item">Element 2</div>
    <div class="flex-item">Element 3</div>
    <div class="flex-item">Element 4</div>
    <div class="flex-item">Element 5</div>
    <div class="flex-item">Element 6</div>
</div>

.flex-container {
    display: flex;
    flex-wrap: wrap;
    gap: 16px;
    padding: 16px;
    background-color: #f9f9f9;
}
.flex-item {
    flex: 1 1 calc(33.333% - 16px);
    background-color: #e6e6fa;
    padding: 16px;
    border: 1px solid #ccc;
    border-radius: 8px;
    text-align: center;
}
@media (max-width: 768px) {
    .flex-item {
        flex: 1 1 calc(50% - 16px);
    }
}
@media (max-width: 480px) {
    .flex-item {
        flex: 1 1 100%;
    }
}

Beschreibung

Das Responsive Flexbox Layout ist vielseitig und einfach anpassbar. Mit Flexbox können Inhalte gleichmäßig verteilt und responsiv gestaltet werden. Es eignet sich ideal für Galerien, Produktanzeigen oder dynamische Layouts.
