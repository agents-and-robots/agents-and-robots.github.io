Fixed Footer Layout

Kurzbeschreibung

Ein Layout mit einem fixierten Fußbereich, der immer am unteren Rand des Bildschirms bleibt. Ideal für Webseiten mit kurzen Inhalten, die eine konstante Navigation oder Informationen im Footer benötigen.

Beispiel

Dies ist der Hauptinhalt der Seite. Scrollen Sie, um den fixierten Footer am unteren Bildschirmrand zu sehen.

HTML und CSS


<body>
    <header>
        <h1>Fixed Footer Layout</h1>
    </header>
    <main>
        <section>
            <p>Dies ist der Hauptinhalt.</p>
        </section>
    </main>
    <footer>
        <p>Dies ist der fixierte Footer.</p>
    </footer>
</body>

body {
    display: flex;
    flex-direction: column;
    height: 100vh;
    margin: 0;
}
main {
    flex: 1;
    padding: 16px;
    background-color: #f9f9f9;
}
footer {
    background-color: #2c3e50;
    color: white;
    text-align: center;
    padding: 16px;
    position: sticky;
    bottom: 0;
    width: 100%;
}

Beschreibung

Das Fixed Footer Layout eignet sich besonders für Webseiten, die wichtige Informationen oder Navigationselemente im Footer unterbringen möchten. Der Footer bleibt immer sichtbar, selbst bei kurzen Seiteninhalten.
