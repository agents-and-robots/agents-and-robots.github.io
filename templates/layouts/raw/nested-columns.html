Nested Columns Layout

Kurzbeschreibung

Ein Layout, das verschachtelte Spalten innerhalb von Zeilen unterstützt. Ideal für komplexe Layouts mit mehreren Ebenen.

Beispiel

Spalte 1
Spalte 2

Spalte 1 mit weiteren verschachtelten Spalten:

Inner 1
Inner 2
Spalte 2

HTML und CSS


<div class="nested-container">
    <div class="nested-row">
        <div class="nested-column">Spalte 1</div>
        <div class="nested-column">Spalte 2</div>
    </div>
    <div class="nested-row">
        <div class="nested-column">
            <p>Spalte 1 mit weiteren verschachtelten Spalten:</p>
            <div class="nested-row">
                <div class="nested-column">Inner 1</div>
                <div class="nested-column">Inner 2</div>
            </div>
        </div>
        <div class="nested-column">Spalte 2</div>
    </div>
</div>

.nested-container {
    display: flex;
    flex-direction: column;
    gap: 16px;
    max-width: 800px;
    margin: 0 auto;
}

.nested-row {
    display: flex;
    gap: 16px;
}

.nested-column {
    flex: 1;
    background-color: #e6e6fa;
    padding: 16px;
    border: 1px solid #ccc;
    border-radius: 8px;
}

Beschreibung

Das Nested Columns Layout ist ideal für komplexe Designs, die Spalten in mehreren Hierarchieebenen erfordern. Es ermöglicht die Erstellung von verschachtelten Strukturen, die flexibel und leicht anpassbar sind.
