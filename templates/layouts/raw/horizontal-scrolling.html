Horizontal Scrolling Layout

Kurzbeschreibung

Ein Layout, das horizontales Scrollen ermöglicht, um Inhalte in einer Reihe anzuzeigen. Ideal für Galerien, Produktvorschauen oder Artikelübersichten.

Beispiel

Item 1
Item 2
Item 3
Item 4
Item 5
Item 6

HTML und CSS


<div class="scroll-container">
    <div class="scroll-item">Item 1</div>
    <div class="scroll-item">Item 2</div>
    <div class="scroll-item">Item 3</div>
    <div class="scroll-item">Item 4</div>
    <div class="scroll-item">Item 5</div>
    <div class="scroll-item">Item 6</div>
</div>

.scroll-container {
    display: flex;
    overflow-x: auto;
    white-space: nowrap;
    padding: 16px;
    background-color: #f9f9f9;
    border: 1px solid #ccc;
}

.scroll-item {
    display: inline-block;
    width: 200px;
    margin-right: 16px;
    background-color: #e6e6fa;
    text-align: center;
    padding: 16px;
    border: 1px solid #ddd;
    border-radius: 8px;
}

Beschreibung

Das Horizontal Scrolling Layout ist ideal, um Inhalte horizontal anzuzeigen. Es ist besonders nützlich für Galerien, Slider oder Bereiche, in denen viele Elemente nebeneinander dargestellt werden sollen, ohne den vertikalen Platz zu beanspruchen.
