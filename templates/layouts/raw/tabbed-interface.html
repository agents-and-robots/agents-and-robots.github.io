Tabbed Interface Layout

Kurzbeschreibung

Ein Layout mit Registerkarten (Tabs), um Inhalte platzsparend und strukturiert darzustellen. Ideal für Anwendungen mit mehreren Ansichten oder Kategorien.

Beispiel

Tab 1

Inhalt für Tab 1.

Tab 2

Inhalt für Tab 2.

Tab 3

Inhalt für Tab 3.

HTML und CSS


<div class="tab">
    <button class="tablinks" onclick="openTab(event, 'Tab1')">Tab 1</button>
    <button class="tablinks" onclick="openTab(event, 'Tab2')">Tab 2</button>
    <button class="tablinks" onclick="openTab(event, 'Tab3')">Tab 3</button>
</div>
<div id="Tab1" class="tabcontent" style="display: block;">
    <h3>Tab 1</h3>
    <p>Inhalt für Tab 1.</p>
</div>
<div id="Tab2" class="tabcontent">
    <h3>Tab 2</h3>
    <p>Inhalt für Tab 2.</p>
</div>
<div id="Tab3" class="tabcontent">
    <h3>Tab 3</h3>
    <p>Inhalt für Tab 3.</p>
</div>

Verwendetes JavaScript


function openTab(evt, tabName) {
    let i, tabcontent, tablinks;
    tabcontent = document.getElementsByClassName("tabcontent");
    for (i = 0; i < tabcontent.length; i++) {
        tabcontent[i].style.display = "none";
    }
    tablinks = document.getElementsByClassName("tablinks");
    for (i = 0; i < tablinks.length; i++) {
        tablinks[i].className = tablinks[i].className.replace(" active", "");
    }
    document.getElementById(tabName).style.display = "block";
    evt.currentTarget.className += " active";
}

Beschreibung

Das Tabbed Interface Layout bietet eine benutzerfreundliche Möglichkeit, verschiedene Inhalte auf einer Seite zu organisieren. Es eignet sich hervorragend für Dashboards, Einstellungen oder umfangreiche Inhalte, die logisch in Kategorien unterteilt werden können.
