Accordion Layout

Kurzbeschreibung

Ein interaktives Layout, bei dem Inhalte ein- und ausgeklappt werden können. Perfekt für FAQs oder Inhalte, die kompakt dargestellt werden sollen.

Beispiel

Inhalt des ersten Abschnitts.

Inhalt des zweiten Abschnitts.

Inhalt des dritten Abschnitts.

HTML und CSS


<button class="accordion" onclick="toggleAccordion(event)">Abschnitt 1</button>
<div class="panel">
    <p>Inhalt des ersten Abschnitts.</p>
</div>
<button class="accordion" onclick="toggleAccordion(event)">Abschnitt 2</button>
<div class="panel">
    <p>Inhalt des zweiten Abschnitts.</p>
</div>
<button class="accordion" onclick="toggleAccordion(event)">Abschnitt 3</button>
<div class="panel">
    <p>Inhalt des dritten Abschnitts.</p>
</div>

Verwendetes JavaScript


function toggleAccordion(event) {
    const content = event.currentTarget.nextElementSibling;
    if (content.style.maxHeight) {
        content.style.maxHeight = null;
    } else {
        content.style.maxHeight = content.scrollHeight + "px";
    }
}

Beschreibung

Das Accordion Layout ist ideal, um Inhalte kompakt und interaktiv zu präsentieren. Benutzer können durch Klicken auf Überschriften zusätzliche Inhalte ein- oder ausblenden, was es besonders für FAQs, Anleitungen oder ähnliche Inhalte geeignet macht.
