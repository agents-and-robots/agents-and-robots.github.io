Two Column - Equal Width (Zwei Spalten, gleiche Breite)

Kurzbeschreibung

Ein Layout mit zwei Spalten, die gleich breit sind. Dieses Layout ist vielseitig einsetzbar und eignet sich für Inhalte, die gleichwertig nebeneinander dargestellt werden sollen.

HTML und CSS


<div style="display: flex; max-width: 800px; margin: 0 auto; border: 1px solid #ccc;">
    <div style="flex: 1; background-color: #f0f8ff; padding: 16px;">
        <h2>Spalte 1</h2>
        <p>Dies ist der Inhalt der ersten Spalte.</p>
    </div>
    <div style="flex: 1; background-color: #faebd7; padding: 16px;">
        <h2>Spalte 2</h2>
        <p>Dies ist der Inhalt der zweiten Spalte.</p>
    </div>
</div>

Beschreibung

Das Two Column Layout mit gleicher Breite ist ideal, wenn zwei Inhalte gleicher Wichtigkeit nebeneinander präsentiert werden sollen. Es ist besonders nützlich für Vergleichstabellen, parallel dargestellte Inhalte oder Designs, die eine symmetrische Optik erfordern.
