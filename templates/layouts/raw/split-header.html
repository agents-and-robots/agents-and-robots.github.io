Split Header Layout

Kurzbeschreibung

Ein Layout mit einer geteilten Kopfzeile, die ein Logo, eine Navigation und eine Aktion wie einen Login-Button enthält. Perfekt für Webseiten mit klarer Struktur und einfacher Navigation.

Beispiel

Logo

  • Home
  • About
  • Services
  • Contact

HTML und CSS


<header style="display: flex; justify-content: space-between; align-items: center; padding: 16px; background-color: #f0f8ff;">
    <div style="flex: 1;">
        <h2 style="margin: 0;">Logo</h2>
    </div>
    <nav style="flex: 2; text-align: center;">
        <ul style="list-style: none; margin: 0; padding: 0; display: flex; justify-content: center; gap: 16px;">
            <li><a href="#" style="text-decoration: none;">Home</a></li>
            <li><a href="#" style="text-decoration: none;">About</a></li>
            <li><a href="#" style="text-decoration: none;">Services</a></li>
            <li><a href="#" style="text-decoration: none;">Contact</a></li>
        </ul>
    </nav>
    <div style="flex: 1; text-align: right;">
        <button style="padding: 8px 16px; background-color: #007BFF; color: white; border: none; border-radius: 4px; cursor: pointer;">Login</button>
    </div>
</header>

Beschreibung

Das Split Header Layout eignet sich hervorragend für Webseiten, die eine klare Trennung von Branding, Navigation und Aktionen in der Kopfzeile benötigen. Es ist besonders nützlich für Seiten mit umfangreichen Menüs oder Call-to-Action-Elementen.
