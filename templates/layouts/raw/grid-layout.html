Grid Layout (Gitterlayout)

Kurzbeschreibung

Ein flexibles Layout, das Inhalte in einem Raster organisiert, ideal für Übersichtsseiten oder Galerien.

Beispiel

Element 1

Inhalt des ersten Elements.

Element 2

Inhalt des zweiten Elements.

Element 3

Inhalt des dritten Elements.

Element 4

Inhalt des vierten Elements.

Element 5

Inhalt des fünften Elements.

Element 6

Inhalt des sechsten Elements.

Beschreibung

Das Grid Layout eignet sich besonders für die Darstellung von Elementen in einem geordneten Raster. Es kann für Galerien, Portfolios oder beliebige Anwendungen, die eine übersichtliche Anordnung benötigen, genutzt werden.
