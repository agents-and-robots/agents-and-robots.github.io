Header with Sticky Navigation

  • Home
  • Über uns
  • Kontakt

Kurzbeschreibung

Ein Layout mit einer fixierten Kopfzeile, die beim Scrollen sichtbar bleibt. Ideal für Navigationsmenüs oder Branding.

Beispiel

Scrollen Sie nach unten, um die fixierte Navigation zu sehen.

Dies ist ein Beispieltext, der genügend Höhe bietet, um das Scrollverhalten zu demonstrieren.

HTML und CSS


<header style="position: sticky; top: 0; background-color: #2c3e50; color: white; padding: 16px; z-index: 1000;">
    <h1>Header with Sticky Navigation</h1>
    <nav>
        <ul style="display: flex; list-style: none; margin: 0; padding: 0;">
            <li style="margin-right: 16px;"><a href="#" style="color: white; text-decoration: none;">Home</a></li>
            <li style="margin-right: 16px;"><a href="#" style="color: white; text-decoration: none;">Über uns</a></li>
            <li><a href="#" style="color: white; text-decoration: none;">Kontakt</a></li>
        </ul>
    </nav>
</header>
<main style="padding: 16px; background-color: #ecf0f1;">
    <p>Scrollen Sie nach unten, um die fixierte Navigation zu sehen.</p>
    <div style="height: 2000px; padding: 16px;">
        <p>Dies ist ein Beispieltext, der genügend Höhe bietet, um das Scrollverhalten zu demonstrieren.</p>
    </div>
</main>

Beschreibung

Die Sticky Navigation ist eine effektive Möglichkeit, wichtige Menüpunkte jederzeit zugänglich zu machen. Sie eignet sich besonders gut für lange Seiten oder Webanwendungen mit häufigem Navigationsbedarf.
