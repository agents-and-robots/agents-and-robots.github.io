Full Height Layout

Kurzbeschreibung

Ein Layout, das den gesamten verfügbaren vertikalen Platz ausfüllt und Inhalte zentral ausrichtet. Perfekt für Landing Pages oder wichtige Mitteilungen.

Beispiel

Inhalt ist vertikal und horizontal zentriert.

HTML und CSS


<body>
    <header>
        <h1>Full Height Layout</h1>
    </header>
    <main style="height: 100vh; display: flex; justify-content: center; align-items: center; background-color: #f9f9f9;">
        <p>Inhalt ist vertikal und horizontal zentriert.</p>
    </main>
    <footer>
        <p>Footer-Inhalt</p>
    </footer>
</body>

body {
    margin: 0;
    font-family: Arial, sans-serif;
    display: flex;
    flex-direction: column;
    min-height: 100vh;
}
header, footer {
    background-color: #2c3e50;
    color: white;
    text-align: center;
    padding: 16px;
}
main {
    flex: 1;
    display: flex;
    justify-content: center;
    align-items: center;
    background-color: #f9f9f9;
    padding: 16px;
}

Beschreibung

Das Full Height Layout eignet sich hervorragend für Designs, die Inhalte in den Fokus rücken möchten. Es ist ideal für minimalistische Designs oder Anwendungen mit zentral ausgerichteten Inhalten.
