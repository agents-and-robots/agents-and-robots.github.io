Sidebar with Dropdown Menu

Kurzbeschreibung

Ein Layout mit einer Seitenleiste und Dropdown-Menüs zur Organisation von Inhalten.

Beispiel

Menü

Unterpunkt 1 Unterpunkt 2 Unterpunkt 3
Unterpunkt 1 Unterpunkt 2
Unterpunkt 1

Dies ist der Hauptinhalt der Seite.

HTML und CSS


<div class="sidebar" style="width: 250px; background-color: #2c3e50; color: white; padding: 16px;">
    <h2>Menü</h2>
    <button class="menu-item" onclick="toggleDropdown(event)">Option 1</button>
    <div class="dropdown" style="display: none; padding-left: 16px;">
        <a href="#" style="color: white; text-decoration: none; display: block; padding: 5px 0;">Unterpunkt 1</a>
        <a href="#" style="color: white; text-decoration: none; display: block; padding: 5px 0;">Unterpunkt 2</a>
        <a href="#" style="color: white; text-decoration: none; display: block; padding: 5px 0;">Unterpunkt 3</a>
    </div>
</div>

Verwendetes JavaScript


function toggleDropdown(event) {
    const dropdown = event.currentTarget.nextElementSibling;
    if (dropdown.style.display === "block") {
        dropdown.style.display = "none";
    } else {
        dropdown.style.display = "block";
    }
}

Beschreibung

Das Sidebar mit Dropdown-Layout ist eine effiziente Möglichkeit, umfangreiche Menüs in einer Seitenleiste zu strukturieren. Benutzer können Untermenüs ein- oder ausklappen, um schnellen Zugriff auf Inhalte zu erhalten, während die Hauptnavigation übersichtlich bleibt.
